# Directives

## Tips

+ Use kind of "namespace" prefix

## Types of Directives

+ Components (Widget)
+ Decorators (ng-click, ng-show... Don't have template and add functionality)
+ Strutural/Templating (e.g. ng-repeat; manipulates the DOM)

## Types of Variables

+ = (input/output)
+ @ (only input)
+ & (for the methods)
